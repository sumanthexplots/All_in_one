# All_in_one
🌐 IP Lookup Tool

A fast and simple Python-based IP Lookup utility that retrieves detailed information about any public IP address.

🚀 Installation

pkg update && pkg upgrade -y
pkg install git -y
pkg install python -y

📥 Clone Repository

git clone https://github.com/sumanthexplots/All_in_one.git

cd All_in_one

▶️ Run The Tool

python3 sumanth5.py

✨ Features

- 🌍 IP Geolocation
- 🏙️ Country & City Detection
- 📡 ISP Information
- 🛰️ Timezone Details
- ⚡ Fast Response
- 🐍 Python Based

📌 Requirements

- Python 3.x
- Internet Connection
